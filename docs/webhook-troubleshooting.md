# Webhook Module Troubleshooting

## Quick Verification Steps

### 1. Check Autotask Webhook Registration
```bash
source <(grep -v '^#' .secrets/.env | sed 's/^/export /') && \
curl -s -X POST \
  -H "ApiIntegrationCode: ${AUTOTASK_INTEGRATION_CODE}" \
  -H "UserName: ${AUTOTASK_USERNAME}" \
  -H "Secret: ${AUTOTASK_SECRET}" \
  -H "Content-Type: application/json" \
  -d '{"filter": []}' \
  "https://webservices5.autotask.net/ATServicesRest/V1.0/TicketNoteWebhooks/query" | \
jq '.items[] | {id, name, webhookUrl, isActive, isSubscribedToCreateEvents, isSubscribedToUpdateEvents, isSubscribedToDeleteEvents}'
```

### 2. Check Make Webhook Status
```bash
source <(grep -v '^#' .secrets/.env | sed 's/^/export /') && \
curl -s -H "Authorization: Token $MAKE_API_KEY" \
  "$MAKE_API_URL/scenarios/YOUR_SCENARIO_ID" | \
jq '{hookId, isActive, lastEdit}'
```

### 3. Verify Webhook Connectivity
```bash
# Get webhook URL from scenario, then:
curl -s -H "Authorization: Token $MAKE_API_KEY" \
  "$MAKE_API_URL/hooks/WEBHOOK_ID/ping" | \
jq '{attached, learning, gone, address}'
```

## Common Issues

### Issue: "No webhook registered in Autotask"
- **Cause:** Module deployment failed or connection invalid
- **Fix:** Re-save webhook module in Make scenario
- **Verify:** Run check #1 above

### Issue: "Webhook exists but scenario shows hookId: null"
- **Cause:** Make scenario disconnected from webhook
- **Fix:** Delete and re-add webhook module
- **Verify:** Run check #2 above

### Issue: "Events not triggering"
- **Cause:** Webhook fields not configured in Autotask
- **Fix:** Add TicketNoteWebhookFields for desired fields
- **Note:** Currently auto-configures basic fields

### Issue: "Authentication errors in Make logs"
- **Cause:** Invalid Autotask connection or expired credentials
- **Fix:** Refresh/recreate Autotask connection in Make

## Module Configuration Reference

### Event Types Mapping:
- **"create"** → `isSubscribedToCreateEvents: true`
- **"update"** → `isSubscribedToUpdateEvents: true`
- **"delete"** → `isSubscribedToDeleteEvents: true`
- **Empty array** → All events enabled

### Generated Webhook Name:
- Format: `"Make Webhook - {hook.name}"`
- Example: `"Make Webhook - Ticket Note Events"`

### Secret Key:
- Auto-generated by Make: `{{hook.secretKey}}`
- Used for HMAC-SHA1 signature validation

## Cleanup Process

When webhook module is removed:
1. Make calls cleanup DELETE endpoint
2. Autotask webhook registration is removed
3. No orphaned webhooks should remain

If cleanup fails, manually delete with:
```bash
curl -X DELETE \
  -H "ApiIntegrationCode: ${AUTOTASK_INTEGRATION_CODE}" \
  -H "UserName: ${AUTOTASK_USERNAME}" \
  -H "Secret: ${AUTOTASK_SECRET}" \
  "https://webservices5.autotask.net/ATServicesRest/V1.0/TicketNoteWebhooks/WEBHOOK_ID"
```
